PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics I - Differential Calculus
  • Engineering Physics - Optics, Waves
  • Basic Electrical - Circuits & Networks
  • Engineering Mechanics - Forces & Motion

Second Year (SE)

  • Data Structures - Linked Lists, Trees
  • OOP using C++ - Classes, Inheritance
  • Digital Electronics - Logic Gates, Circuits
  • Computer Networks - Protocols, OSI model

Third Year (TE)

  • DBMS - SQL, ER Diagrams
  • Operating Systems - Processes, Scheduling
  • Machine Learning - Classification, Regression
  • Software Engineering - SDLC, Testing

Final Year (BE)

  • AI & Deep Learning - CNN, RNN
  • Cloud Computing - AWS, Virtualization
  • Cybersecurity - Encryption, Firewalls
  • Project Phase I & II - Final Year Project
